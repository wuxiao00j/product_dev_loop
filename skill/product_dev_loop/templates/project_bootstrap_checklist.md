# Project Bootstrap Checklist

## 新建项目前要确认

- 项目名是否明确
- 当前目标是否明确
- 是否已有参考产品或已有仓库
- 是否存在明确非目标

## 建目录时要准备

- 独立项目目录
- `PRD.md`
- `PROJECT_STATE.md`
- `PROGRESS.md`
- `IMPLEMENTATION_NOTES.md`
- `CHANGE_PROPOSALS.md`
- `IMPLICIT_REQUIREMENTS.md`
- `DECISIONS.md`
- `TEST_GUIDANCE.md`
- `PROMPTS/`

## PRD 前要准备

- 当前问题定义
- 核心场景
- 范围边界
- 验收标准草案
- 已识别但未确认的隐性要求

## 第一次派单前要检查

- PRD 是否存在
- PRD 状态是否明确
- 本轮目标是否足够小
- 必读文件是否齐
- 测试入口是否能描述
- 多项目是否已隔离
